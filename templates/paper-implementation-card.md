# Paper Implementation Card

Use this for every serious paper implementation. The companion paper-workflow PDF is the source for this template, and the primary roadmap decides whether the paper belongs in the active milestone sequence.

```markdown
# Paper implementation card

Paper:
URL:
Category:
Core mechanism:
Why it matters for model building:
Why it matters for ARC / skill acquisition:
Implementation level: L1 / L2 / L3 / L4
Minimal implementation:
Dataset / environment:
Baseline:
Metric:
Expected failure:
Kill criterion:
Integration target:
Public artifact:
Decision: integrate / archive / reject
```

The most important fields are:

- `Metric`
- `Kill criterion`
- `Integration target`

A paper implementation without those three fields is usually just paper tourism.
