# Notebooks

This directory is the notebook-first execution layer for the learning plan.

## Structure

- `templates/`
  - reusable notebook templates
- `week-01/`
  - starter notebooks for the first 7 days

## How To Use

For each day:

1. Read the matching plan in `docs/week-01/`.
2. Open the matching notebook in `notebooks/week-01/`.
3. Work in the notebook first.
4. Save key conclusions back to your notes.
5. End by drafting the daily X post.

## Naming

The notebooks mirror the day-plan names so the workflow stays predictable.
