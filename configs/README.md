# Configs

Configuration files live here and should be committed with every reproducible run.

Suggested subdirectories:

- `model/` for architecture and checkpoint configs.
- `train/` for optimizer, scheduler, batch, and run configs.
- `eval/` for benchmark and scoring configs.
- `serve/` for inference and serving configs.
- `arc/` for ARC-AGI-3 runner and agent configs.
